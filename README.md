# 📖 Descripción

**Mueblería González** es una aplicación web moderna orientada a la exhibición digital de muebles artesanales, administración de productos y gestión de favoritos para clientes.

El sistema utiliza una arquitectura frontend basada en JavaScript moderno conectada a servicios en la nube mediante Firebase y Supabase.

> [!NOTE]
> El proyecto fue diseñado para ofrecer una experiencia rápida, elegante y completamente responsiva para dispositivos móviles, tabletas y computadoras.

> [!IMPORTANT]
> No requiere backend tradicional en PHP ni servidores propios para la administración principal de datos.
